============================================================
           FICHE TECHNIQUE - PREMIUM SPIN & WIN
============================================================

1. PRÉSENTATION DU PROJET
-------------------------
Nom : Premium Spin & Win
Version : 2.1 (Premium Analytics & Luxury UI Edition)
Type : Application Web de Marketing Interactive (Gamification)
Objectif : Capture de leads à haute valeur, intelligence utilisateur avancée et distribution de récompenses premium.

2. ARCHITECTURE TECHNIQUE
-------------------------
- Frontend : HTML5, CSS3 (Advanced Variables & Keyframes), JavaScript (ES6+)
- Graphismes : Canvas 2D (Wheel, Particle System, Dynamic Orbs)
- Audio : Web Audio API (Procedural SFX generation)
- Analytics : Chart.js (Timeline & Device distribution)
- Persistance : LocalStorage (Structured Namespacing)
- APIs Externes : FreeIPAPI (Geolocation)
- Design : High-End Glassmorphism, Luxury Dark/Light Palette, Staggered Animations

3. FONCTIONNALITÉS CLIENT (index.html / script.js)
-------------------------------------------------
- Premium Aesthetic : Nouveau design "Luxury Gaming" avec effets de flou avancés.
- Enhanced Tracking :
    * Détection intelligente du Navigateur & OS (parsing User Agent).
    * Identification du type de terminal (Mobile, Tablette, Desktop).
    * Capture du Referrer et de la Résolution d'écran.
- Win Attribution : Chaque gain est désormais lié directement au lead capturé dans la base de données locale.
- Feedback Visuel : Système de confettis et d'animations d'entrée synchronisées.

4. PANNEAU D'ADMINISTRATION (panel.html / panel.js)
--------------------------------------------------
- Intelligence Dashboard :
    * KPI Cards : Total Leads, Taux de Conversion, Top Pays, Gain le plus fréquent.
    * Visualisations : Graphique temporel des captures et répartition des terminaux.
- Gestion des Données :
    * Tableau interactif avec tri (Date, Email, Localisation) et recherche en temps réel.
    * Filtres avancés par Pays et Type de terminal.
    * Modal "User Intelligence" : Vue détaillée profilée de chaque lead.
- Exportation Multiformat : Support JSON et CSV pour intégration CRM.
- Configuration Dynamique : Édition complète de la roue (poids, couleurs, codes, emojis).

5. LOGIQUE DE CALCUL
--------------------
- Probabilité Pondérée : Algorithme de sélection cumulative pour garantir la rareté des gros lots.
- Device Detection : Logique heuristique basée sur les patterns du User Agent.
- Conversion Tracking : Ratio Leads Uniques / Total Entrées (Simulation session).

6. STRUCTURE DES FICHIERS
-------------------------
/spin-win
├── index.html       # Client Premium Interface
├── script.js        # Engine, Audio & Tracking Logic
├── style.css        # Luxury Design System
├── panel.html       # Advanced Analytics Dashboard
├── panel.js         # Analytics, Filtering & Config Management
├── assets/          # Visual Identity Assets
└── fiche-technique.txt # Cette documentation

7. SÉCURITÉ & PRIVACY
---------------------
- Accès Admin : Authentification locale sécurisée.
- Privacy-First : Toutes les données restent dans le navigateur de l'administrateur.
- Performance : Optimisé pour charger en moins d'une seconde sur connexions 4G.
